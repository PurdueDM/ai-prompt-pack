# Business Prompts — 20 Templates

## Market Analysis

### 1. Competitive Landscape Analysis
```
Analyze the competitive landscape for [INDUSTRY/PRODUCT]. Identify the top 5 competitors by market share. For each, provide: core value proposition, pricing model, key differentiators, known weaknesses, and estimated annual revenue range. Present as a comparison matrix table, then summarize the top 3 opportunities for a new entrant.
```

### 2. TAM/SAM/SOM Market Sizing
```
Calculate the Total Addressable Market (TAM), Serviceable Addressable Market (SAM), and Serviceable Obtainable Market (SOM) for [PRODUCT/SERVICE] targeting [GEOGRAPHY]. Use a top-down approach with publicly available data points. Show your math, cite reasonable assumptions, and present the result as a funnel diagram in text format.
```

### 3. SWOT Analysis Generator
```
Perform a SWOT analysis for [COMPANY/PRODUCT]. Context: [2-3 sentences about the business]. For each quadrant, provide exactly 5 items ranked by impact. Then generate a 3-point action plan that leverages strengths against opportunities and mitigates the intersection of weaknesses and threats.
```

### 4. Customer Persona Builder
```
Create 3 detailed customer personas for [PRODUCT/SERVICE]. For each persona include: name, age range, job title, annual income, goals (3), pain points (3), preferred channels, buying triggers, objections, and a "day in the life" narrative (100 words). Format as persona cards.
```

## Strategy & Planning

### 5. OKR Framework Generator
```
Generate quarterly OKRs for a [TEAM SIZE]-person [DEPARTMENT] team at a [STAGE] [INDUSTRY] company. Current priorities: [LIST 3 PRIORITIES]. Create 3 Objectives with 3 Key Results each. Every Key Result must be measurable with a specific numeric target. Include a confidence score (0.3-0.7 recommended) for each KR.
```

### 6. Go-to-Market Strategy
```
Design a go-to-market strategy for [PRODUCT] launching in [MARKET]. Budget: [AMOUNT]. Timeline: [MONTHS]. Include: positioning statement, channel strategy (ranked by expected ROI), first 90-day milestones, key metrics to track, and a risk register with 5 risks and mitigations.
```

### 7. Pricing Strategy Analysis
```
Recommend a pricing strategy for [PRODUCT/SERVICE] in the [INDUSTRY] market. Current competitors charge [RANGE]. Our cost basis is [COST]. Analyze 3 pricing models (cost-plus, value-based, competitive) and recommend one with justification. Include pricing tiers, anchor pricing suggestions, and expected conversion rate impact.
```

### 8. Partnership Evaluation Framework
```
Evaluate whether [COMPANY A] should partner with [COMPANY B]. Criteria: strategic alignment, resource complementarity, cultural fit, financial impact, risk exposure. Score each criterion 1-10 with justification. Provide a final recommendation with conditions, deal structure suggestion, and exit clause recommendations.
```

## Financial Analysis

### 9. Unit Economics Calculator
```
Calculate the unit economics for [BUSINESS MODEL]. Inputs: Customer Acquisition Cost = [CAC], Average Revenue Per User = [ARPU], Gross Margin = [%], Churn Rate = [%/month]. Calculate: LTV, LTV:CAC ratio, payback period, and breakeven timeline. Flag any metrics outside healthy ranges and suggest specific improvements.
```

### 10. Revenue Projection Model
```
Project monthly revenue for [BUSINESS] over the next 12 months. Current MRR: [AMOUNT]. Growth inputs: organic growth rate [%], paid acquisition budget [$/month], average conversion rate [%], average deal size [$]. Model 3 scenarios: conservative (0.7x), base (1x), aggressive (1.5x). Present as monthly table with cumulative totals.
```

### 11. Cash Flow Sensitivity Analysis
```
Analyze cash flow sensitivity for [BUSINESS]. Current monthly burn: [AMOUNT]. Revenue: [AMOUNT]. Runway: [MONTHS]. Test sensitivity to: ±20% revenue variance, ±30% expense variance, 60-day payment delay, and loss of top customer (contributing [%] of revenue). Identify the breakpoint scenario and recommend a trigger for contingency plan activation.
```

### 12. Investment Memo Generator
```
Write a 1-page investment memo for [COMPANY] raising [AMOUNT] at [VALUATION]. Include: thesis (2 sentences), market opportunity, team strengths, traction metrics, key risks, use of funds breakdown (pie chart format), and comparable exits in the sector. Tone: objective and data-driven.
```

## Operations

### 13. Process Optimization Audit
```
Audit the [PROCESS NAME] workflow that currently takes [TIME] and involves [STEPS] steps across [TEAMS]. Identify bottlenecks, redundant steps, and automation opportunities. Propose an optimized workflow with expected time savings (%). Include an implementation priority matrix (effort vs. impact).
```

### 14. Vendor Selection Scorecard
```
Create a vendor selection scorecard for [SERVICE TYPE]. We're evaluating [VENDOR 1], [VENDOR 2], and [VENDOR 3]. Criteria: cost (25%), capability (30%), reliability (20%), scalability (15%), support (10%). For each vendor, score 1-10 per criterion with justification. Calculate weighted scores and provide a recommendation with caveats.
```

### 15. Risk Register Builder
```
Build a risk register for [PROJECT/INITIATIVE]. Identify 10 risks across categories: technical, financial, operational, regulatory, and market. For each: description, probability (1-5), impact (1-5), risk score, owner, mitigation strategy, contingency plan, and trigger indicator. Sort by risk score descending.
```

### 16. Quarterly Business Review Template
```
Generate a QBR presentation outline for [TEAM/DEPARTMENT]. Last quarter's goals: [LIST]. This quarter's priorities: [LIST]. Include sections: executive summary, KPI dashboard (with RAG status), wins, misses with root cause, customer/stakeholder feedback themes, resource needs, and next quarter roadmap. Format as slide-by-slide outline.
```

## Communication

### 17. Executive Summary Writer
```
Write a 200-word executive summary of [DOCUMENT/REPORT/SITUATION]. The audience is [C-SUITE/BOARD/INVESTORS]. Key points to emphasize: [3 POINTS]. Tone: [FORMAL/CONVERSATIONAL]. Include a clear ask or recommendation in the final sentence. Avoid jargon. Every sentence must add information value.
```

### 18. Stakeholder Update Email
```
Draft a stakeholder update email for [PROJECT]. Status: [ON TRACK/AT RISK/DELAYED]. Key updates: [3 BULLET POINTS]. Blockers: [LIST OR NONE]. Next milestones: [LIST WITH DATES]. Tone: transparent and professional. Keep under 200 words. Include a clear call-to-action if decisions are needed.
```

### 19. Meeting Agenda & Pre-Read
```
Create a meeting agenda and pre-read for a [DURATION]-minute [MEETING TYPE] with [PARTICIPANTS]. Objective: [GOAL]. Include: agenda items with time allocations, pre-read materials list, discussion questions, decision items (with options and recommendation), and parking lot for out-of-scope items.
```

### 20. Decision Document (RFC)
```
Write a Request for Comments (RFC) document for [DECISION]. Context: [WHY THIS DECISION IS NEEDED]. Options: [OPTION A], [OPTION B], [OPTION C]. For each option: pros, cons, cost, timeline, and reversibility score. Recommendation: [YOUR PICK] with reasoning. Include: decision criteria, stakeholders impacted, rollback plan, and deadline for comments.
```
