# Content Creation Prompts — 20 Templates

## Blog & Long-Form

### 1. Technical Blog Post Writer
```
Write a 1500-word technical blog post about [TOPIC]. Target audience: [DEVELOPERS/DEVOPS/DATA SCIENTISTS/etc.]. Experience level: [JUNIOR/MID/SENIOR]. Structure: hook (2 sentences), problem statement, solution walkthrough with code examples, benchmarks/results, conclusion with next steps. Tone: authoritative but accessible. Include 3 subheadings optimized for SEO.
```

### 2. How-To Tutorial Generator
```
Write a step-by-step tutorial: "How to [ACCOMPLISH TASK] using [TOOL/TECHNOLOGY]". Prerequisites: [LIST]. Target reader: [PERSONA]. Include: (1) introduction explaining why this matters, (2) numbered steps with code snippets, (3) expected output at each step, (4) troubleshooting section with 3 common errors, (5) "what's next" section with 3 follow-up topics.
```

### 3. Case Study Writer
```
Write a case study about [PROJECT/IMPLEMENTATION]. Client industry: [INDUSTRY]. Challenge: [DESCRIBE]. Solution: [DESCRIBE]. Results: [METRICS]. Structure: Executive Summary (50 words), Challenge (150 words), Approach (200 words), Implementation (300 words), Results (150 words), Lessons Learned (100 words). Tone: professional, data-driven. Do NOT fabricate any metrics — use only the data provided.
```

### 4. Thought Leadership Article
```
Write a thought leadership article on [TREND/TOPIC] for [PUBLICATION TYPE — LinkedIn, Medium, industry blog]. Angle: [YOUR UNIQUE PERSPECTIVE]. Length: 1000 words. Include: contrarian opening that challenges conventional wisdom, 3 supporting arguments with evidence, practical implications, and a forward-looking conclusion. Avoid clichés and buzzword soup.
```

## Social Media

### 5. LinkedIn Post Generator
```
Write a LinkedIn post about [TOPIC/EXPERIENCE/INSIGHT]. Goal: [ENGAGEMENT/AUTHORITY/LEAD GEN]. Tone: [PROFESSIONAL/CONVERSATIONAL/PROVOCATIVE]. Format: hook line (stops the scroll), 3-5 short paragraphs (2-3 sentences each), call to action. Include 3-5 relevant hashtags. Total length: 150-200 words. Avoid: corporate jargon, humble bragging, "I'm excited to announce."
```

### 6. Twitter/X Thread Creator
```
Create a [NUMBER]-tweet thread about [TOPIC]. Hook tweet must create curiosity without clickbait. Each tweet: max 280 characters, one clear idea, flows logically to the next. Include: numbered tweets (1/N format), a knowledge bomb in tweet 3-4, a practical takeaway in the second-to-last, and a CTA in the final tweet. End with a retweet hook back to tweet 1.
```

### 7. Social Media Content Calendar
```
Create a 2-week social media content calendar for [BRAND/PERSON] on [PLATFORMS]. Niche: [TOPIC AREA]. Goals: [AWARENESS/ENGAGEMENT/LEADS]. For each post: date, platform, content type (text/image/video/carousel), topic, draft copy (50-100 words), best posting time, and 3 hashtags. Include 2 engagement posts, 2 educational posts, 1 promotional post per week.
```

### 8. Video Script Outline
```
Write a script outline for a [DURATION]-minute [YouTube/TikTok/Instagram] video about [TOPIC]. Target audience: [DESCRIBE]. Include: hook (first 5 seconds), intro (who you are, what they'll learn), 3-5 main sections with talking points, B-roll suggestions for each section, transitions, CTA placement, and outro. Note retention danger zones and how to keep attention.
```

## Email Marketing

### 9. Email Sequence Builder
```
Create a [NUMBER]-email sequence for [GOAL — onboarding, nurture, launch, re-engagement]. Product/Service: [DESCRIBE]. Target: [PERSONA]. For each email: subject line (A/B options), preview text, body copy (150-250 words), CTA button text, send timing (delay from previous). Include personalization tokens where appropriate. Sequence should tell a cohesive story.
```

### 10. Cold Outreach Email
```
Write a cold outreach email to [RECIPIENT ROLE] at [COMPANY TYPE]. Purpose: [BOOK MEETING/INTRO PRODUCT/PARTNERSHIP]. My value prop: [1-2 SENTENCES]. Email constraints: under 100 words, no attachments mentioned, personalized opening that references something specific about them. Include 3 variations: direct, curiosity-based, and social-proof-based. Add a follow-up email for day 3 if no response.
```

### 11. Newsletter Template
```
Write a newsletter edition for [BRAND/PUBLICATION]. Topic: [THIS WEEK'S THEME]. Sections: (1) Editor's note — 50 words, personal, sets the frame, (2) Main story — 200 words, the deep insight, (3) Quick hits — 3 news items, 30 words each, (4) Resource of the week — 50 words with link placeholder, (5) CTA — 25 words. Total: ~500 words. Tone: [CASUAL/PROFESSIONAL/WITTY].
```

## SEO & Copywriting

### 12. SEO Content Brief
```
Create an SEO content brief for the keyword "[TARGET KEYWORD]". Include: (1) search intent analysis (informational/transactional/navigational), (2) recommended title tag and meta description, (3) H1 and H2 structure (5-8 headings), (4) key topics to cover (based on top-ranking pages), (5) related keywords to include (10), (6) recommended word count, (7) internal/external linking suggestions, (8) featured snippet opportunity and format.
```

### 13. Landing Page Copy
```
Write landing page copy for [PRODUCT/SERVICE]. Target user: [PERSONA]. Primary CTA: [ACTION]. Structure: (1) hero headline — benefit-focused, max 10 words, (2) subheadline — what it is + who it's for, max 20 words, (3) 3 feature blocks (icon + headline + 2-sentence description), (4) social proof section (testimonial format placeholders), (5) FAQ (5 questions), (6) final CTA with urgency element. Tone: [CASUAL/PROFESSIONAL/PLAYFUL].
```

### 14. Product Description Writer
```
Write [NUMBER] product descriptions for [PRODUCT CATEGORY]. Each description: (1) headline emphasizing key benefit, (2) 80-word description covering features, benefits, and ideal user, (3) 3 bullet points (scannable format), (4) SEO keyword integration: [LIST KEYWORDS]. Tone: [BRAND VOICE]. Avoid superlatives unless backed by data. E-commerce platform: [AMAZON/SHOPIFY/CUSTOM].
```

### 15. Ad Copy Generator
```
Write ad copy for [PRODUCT/SERVICE] on [PLATFORM — Google, Meta, LinkedIn]. Campaign goal: [AWARENESS/TRAFFIC/CONVERSIONS]. Budget tier: [LOW/MED/HIGH]. For each platform create 3 variations: (1) headlines (character limits per platform), (2) description/body text, (3) CTA. Include A/B test recommendations. Target audience: [DEMOGRAPHICS + INTERESTS]. Competitive angle: [DIFFERENTIATOR].
```

## Communication & PR

### 16. Press Release Writer
```
Write a press release for [ANNOUNCEMENT — launch, funding, partnership, milestone]. Company: [NAME]. Industry: [SECTOR]. Include: dateline, headline (max 10 words), subheadline, opening paragraph (who, what, when, where, why), 2 body paragraphs with quotes from [SPOKESPERSON TITLE], boilerplate "About [COMPANY]" (50 words), media contact info placeholder. AP Style. Under 500 words.
```

### 17. Customer Testimonial Request
```
Write a testimonial request email to [CUSTOMER NAME] who [USED OUR PRODUCT FOR X]. Include: warm personalized opening referencing their success, specific questions that elicit quotable responses (3 questions), offer to draft it for their approval, and a deadline. Also generate 3 sample testimonial templates they can customize instead of writing from scratch. Keep email under 150 words.
```

### 18. Webinar/Event Promotion
```
Create promotion content for a webinar titled "[TITLE]" on [DATE]. Speaker: [NAME, TITLE]. Topic: [DESCRIBE]. For: [AUDIENCE]. Create: (1) event description (200 words), (2) LinkedIn promotional post, (3) email invitation, (4) 2 reminder emails (1 week, 1 day before), (5) registration page bullet points (5 reasons to attend). Emphasize takeaways over credentials.
```

### 19. Crisis Communication Template
```
Draft a crisis communication response for [SITUATION — data breach, service outage, product recall, PR incident]. Audience: [CUSTOMERS/PUBLIC/EMPLOYEES]. Severity: [HIGH/MEDIUM/LOW]. Include: (1) initial acknowledgment statement (50 words, within 1 hour), (2) detailed update template (200 words, within 4 hours), (3) resolution announcement template (150 words), (4) internal team talking points (5 bullets). Tone: empathetic, transparent, action-oriented.
```

### 20. Content Repurposing Planner
```
I have this piece of content: [PASTE OR DESCRIBE ORIGINAL CONTENT — blog post, talk, report]. Help me repurpose it into 10 derivative pieces: (1) tweet thread, (2) LinkedIn post, (3) email newsletter excerpt, (4) infographic outline, (5) short-form video script, (6) podcast episode outline, (7) slide deck (5 slides), (8) quote graphics (3), (9) FAQ page section, (10) community discussion prompt. For each, provide the adapted content draft.
```
